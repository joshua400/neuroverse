# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 4.1.x   | :white_check_mark: |
| < 4.1   | :x:                |

## Reporting a Vulnerability

We take the security of NeuroVerse seriously. If you believe you have found a security vulnerability, please report it to us by emailing joshuaragiland@gmail.com.

Please do not report security vulnerabilities through public GitHub issues.

We will acknowledge your report within 48 hours and provide an estimated timeframe for a fix. We ask that you do not disclose the vulnerability publicly until we have had a chance to address it.
